# Example MCP Client Prompts

These prompts are intended for an MCP-compatible assistant connected to the NIH Research MCP server.

## Cohort discovery

- Find patients over 65 with AAA diameter greater than 3 cm.
- Which synthetic patients have AAA diameter at least 4.0 cm?
- Return metadata for patient SYN-017.

## Cohort statistics

- Compute AAA prevalence in the synthetic cohort.
- Summarize AAA-positive counts by sex.
- What is the mean and median AAA diameter?

## Literature search

- Find publications about automated AAA detection.
- Search for papers about synthetic DICOM metadata.
- Which publications mention atherosclerotic plaque?

## Protocol lookup

- Search the protocols for DICOM de-identification rules.
- What is the AAA measurement threshold in this demo?
- Find protocol guidance about burned-in annotations.
